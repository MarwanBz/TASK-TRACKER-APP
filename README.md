# task-tracker-app

this is a task tracker app for tracking all your tasks throwout the day 

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
### Run JSON server as backend
```
npm run backend
```
